LostAndFound Documentation
==========================

This documentation describes the Lost and Found desktop system built for
ECM2429. The application uses an MVC-style structure:

- Model: `backend.model` (SQLite persistence and validation)
- View/Controller: `frontend.view` and `frontend.controller` (Tkinter UI)
